About me..

Programming combines my passion for working with technical and creative mediums. I’ve always been curious and interested in how things work, which has led me to learning to code, via building my own PC!
I’m starting off with HTML, CSS and JavaScript – initially with free community developed resources like freeCodeCamp and The Odin Project. Already I’m finding that the problem solving approach and analytical thinking from my Physics Master is kicking in.
My work experience is mainly in the environmental sector, and I would love to eventually put my new skills to use in this sector.
