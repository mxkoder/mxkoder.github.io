About me..

Programming combines my passion for working with technical and creative mediums. I’ve always been curious and interested in how things work, which has led me to learning to code, via building my own PC!


I’m started off with HTML, CSS and JavaScript – initially with free community developed resources like freeCodeCamp and The Odin Project. Already I’m finding that the problem solving approach and analytical thinking from my Physics Masters is kicking in!


In September 2022 I started working as a Trainee Software Engineer in the UK civil service sector, on a programme that involves applied learning while working on teams. I’m excited about the opportunities this brings, and looking forward to contributing to a service that has millions of users.


I’m also completing a 6 month Software Engineer course with Manchester Codes which I started in May 2022, covering:

  • HTML & CSS
  • JavaScript & ES6
  • Git, Github & CLI
  • Test Driven Development
  • React
  • Docker
  • Node & Jest
  • MySQL
  • API development
  • Express.js

I’m enjoying the course – it’s project based, and great to learn as part of a cohort after self-teaching. We do a lot of pair and group programming, and having a clear curriculum with support has accelerated my learning.


The “lift as you climb” (or in my case at the moment, “lift as you learn”!) approach resonates with me, and I’m learning in public sharing my experiences on social media and my blog.


I’m looking forward to keep learning and start my first role in Software Engineering!
