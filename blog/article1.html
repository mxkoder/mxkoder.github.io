Thanks for taking a look here! Apparently it’s good to write while you learn, so here goes. When I started learning to code, I quickly saw that writing was crucial. At first I was worried that I wouldn’t know what to write about, but now I have the opposite problem with too many ideas!

Some ideas of what you’ll be able to find in this blog:

  • About learning to code, how to guides etc– If I don’t find a clear answer to a problem I’m trying to solve online, I’ll write one (that’s if I figure out how to solve the problem!)
  • Diversity and inclusion in tech– Both about ways to increase diversity in tech, and about how tech can be applied in a more inclusive way.
  • Whatever else inspires!
